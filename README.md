## JooDock

화면 상단 중앙에 마우스를 올리면 자주 쓰는 파일에 빠르게 접근할 수 있는 앱입니다.

### 주요 기능
- Hover 트리거 (화면 상단 중앙)
- 시스템 전체 파일 검색
- 최근 파일 자동 표시
- 드래그 앤 드롭 파일 추가
- 파일 그룹핑

### 기술 스택
- **macOS**: SwiftUI + AppKit
- **Windows**: Tauri + React + TypeScript
